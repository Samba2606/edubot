# 🎓 EduBot — AI Study Assistant

An AI-powered study assistant built with Groq API and Llama 3.3 70B.
Live Demo: https://edubot-vyqk.onrender.com

## Features
- Explains any concept in simple terms with examples
- Generates MCQ quizzes with score tracking
- Maintains full conversation history
- Topic suggestion sidebar for quick access
- Quiz mode with performance tracking

## Tech Stack
- Python
- Streamlit
- Groq API (Llama 3.3 70B)
- Prompt Engineering

## How to Run
1. Clone the repository
2. Install dependencies: pip install -r requirements.txt
3. Add your Groq API key as environment variable: GROQ_API_KEY
4. Run: streamlit run app.py
